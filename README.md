# Amazon Sales Excel Dashboard

## project Overview
This project is an interactive Excel dashboard built using Amazon sales data for 2025.The dashboard provides insights into sales performance, product categories, customer locations, and order status.
## Files Included
– Raw dataset

– Excel dashboard with charts & slicers
## Dataset Description
The dataset contains the following columns:
-Order ID

-Date

-Product

-Category

-Price

-Quantity

-Total Sales

-Customer Name

-Customer Location

-Payment Method

-Status

## Dashboard Features
-Total Sales KPI

-Sales by Category (Bar/Pie Chart)

-Monthly Sales Trend (Line Chart)

- Order Status Breakdown

-Top Products by Sales

- Customer Location Analysis
 
-Interactive Filters (Slicers)

 ## Tools Used
-Microsoft Excel

-Pivot Tables

-Pivot Charts

-Slicers

-Data Cleaning

-Formulas (SUM, COUNT, etc.)

## Key Insights
-Identify top-selling products

-Analyze revenue by category

-Track order status (Completed, Pending, Cancelled)

-Understand customer purchase patterns
